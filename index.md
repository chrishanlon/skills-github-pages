# GitHub Markdown Guide

## Headers
Use `#` for headers. You can go up to six levels.

#### Header 4
##### Header 5

## Text Formatting
- **Bold:** `**text**` or `__text__` → **text**
- *Italic:* `*text*` or `_text_` → *text*
- ~~Strikethrough:~~ `~~text~~` → ~~text~~
